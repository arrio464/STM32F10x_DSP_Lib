What's New in STM32F10xxx DSP Library firmware and Demo  V2.0.0?

The STM32F10xxx DSP firmware Library & FFT Demo V2.0.0 is an update of V1.0.1 that makes the FFT demo and library compliant with the Cortex™ microcontroller software interface standard (CMSIS).

What is STM32F10xxx DSP Library firmware and Demo ?

The STM32F10xxx DSP (digital signal processing) library,is a suite of common digital signal processing functions:

  • PID controller
  •  Fast Fourier transform
  •  FIR and IIR filters

The library contains C and assembly functions that have been ported and tested on the IAR
EWARM 5.20, Keil RVMDK 3.23 and Raisonance RIDE7 (GNU compiler) toolchains.).

The package is coming with  a demonstration example for FFT"Fast Fourier Transform".
